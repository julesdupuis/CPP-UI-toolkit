# What is this ?
This project was an attempt to make an UI toolkit in C++.

It is mainly inspired by Java Swing which was the only UI toolkit I knew at the time.

My main goal was to learn more about UI toolkits in general by trying to implement one by myself.

## Compiling
You will need to have clang++, [Raylib](https://github.com/raysan5/raylib) and [CPPUnit](https://www.freedesktop.org/wiki/Software/cppunit/) installed to be able to compile.

You should be able to to compile, run the tests and the program by running `make`.

## Class diagram
You can use PlantUML on `project.puml` to get a class diagram of the project.
